# _legacy — исходные демо-приложения

Здесь хранятся два первоначальных приложения, на основе которых был построен проект «Сверка».
Их логика **уже перенесена** в модули `assets/functions/` и в работающем приложении
не используется. Папка оставлена как референс для будущей доработки.

| Папка | Назначение | Куда перенесено |
|---|---|---|
| `SearchDiff/` | Сравнение Excel-таблиц по суммам компаний | `assets/functions/compare-tables.js` |
| `SearchFall/` | Авансовый отчёт vs Банк (дубли + сравнение) | `assets/functions/advance-vs-bank.js` |

> Эти файлы не влияют на работу основного приложения и не задействованы в GitHub Pages.
