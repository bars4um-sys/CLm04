# 📊 Приложение для сравнения Excel таблиц

Веб-приложение для быстрого сравнения двух Excel таблиц с автоматической агрегацией данных по компаниям.

## 🚀 Возможности

✅ **Загрузка таблиц** - поддержка форматов .xlsx, .xls, .csv  
✅ **Автоматическая агрегация** - суммирование по компаниям  
✅ **Быстрое сравнение** - выявление всех расхождений  
✅ **Удобный интерфейс** - простая визуализация результатов  
✅ **Адаптивный дизайн** - работает на всех устройствах  

## 📋 Как использовать

### 1. Запуск приложения
Откройте файл `index.html` в веб-браузере

### 2. Подготовка данных
Ваши Excel таблицы должны содержать два столбца:
- **Наименование** - название компании (текст)
- **Сумма** - числовое значение

Пример структуры:
```
Наименование          | Сумма
ООО Компания A        | 1000
ООО Компания A        | 500
ООО Компания B        | 2000
```

> **Важно:** По одной компании может быть несколько строк. Приложение автоматически просуммирует все суммы для каждой компании.

### 3. Загрузка таблиц
1. Нажмите кнопку **"Загрузите таблицу 1"** и выберите первый Excel файл
2. Нажмите кнопку **"Загрузите таблицу 2"** и выберите второй Excel файл
3. Имена файлов появятся под кнопками с галочкой ✓

### 4. Сравнение
Нажмите кнопку **"Сравнить таблицы"**

### 5. Просмотр результатов
Приложение отобразит:
- Количество компаний с расхождениями
- Список всех компаний, по которым найдены различия
- Сумму по каждой таблице
- Размер разницы

## 🔧 Технические детали

### Используемые технологии
- **HTML5** - структура страницы
- **CSS3** - стилизация и адаптивность
- **JavaScript (ES6+)** - логика приложения
- **XLSX.js** - парсинг Excel файлов

### Алгоритм работы

1. **Загрузка файлов** - XLSX.js читает Excel/CSV файлы
2. **Агрегация** - данные группируются по названию компании и суммируются
3. **Сравнение** - сравниваются агрегированные значения
4. **Вывод результатов** - компании с расхождениями выводятся отсортированными по алфавиту

### Поддерживаемые форматы
- ✅ .xlsx (Excel 2007+)
- ✅ .xls (Excel 97-2003)
- ✅ .csv (Comma-separated values)

## 💡 Советы

- Убедитесь, что в таблицах используются одинаковые названия компаний
- Обе таблицы должны иметь заголовки (первая строка)
- Суммы должны быть числовыми значениями
- Если нужны другие названия столбцов, приложение автоматически распознает первый текстовый столбец как наименование и первый числовой как сумму

## 🐛 Решение проблем

### "Таблица пуста или не содержит данных"
- Проверьте, что файл содержит данные
- Убедитесь, что это хорошо сформированный Excel файл

### Расхождения не найдены, хотя они есть
- Проверьте совпадение названий компаний (пробелы, регистр)
- Убедитесь, что суммы - это числа, а не текст

### Приложение не загружает файл
- Попробуйте сохранить файл в формате .xlsx
- Убедитесь, что браузер позволяет загружать файлы
