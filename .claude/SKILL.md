## Как работают функции

### Авансовый отчёт vs Банк
1. Загрузить xlsx-файл с авансовыми отчётами (колонки: Сумма, Контрагент, Назначение)
2. Загрузить xlsx-файл с банковской выпиской (колонки: Дата, Сумма, Назначение платежа)
3. Нажать «Сверить» — сервис находит совпадения и расхождения

### Сравнение таблиц
1. Загрузить два xlsx-файла с одинаковой структурой
2. Выбрать ключевую колонку для сравнения
3. Сервис находит строки, которые есть в первой таблице, но отсутствуют во второй, и наоборот

## Как вносить изменения

- Основная логика — в `assets/functions/*.js` каждая функция независима
- Общие утилиты — `assets/js/utils.js`
- Стили — `assets/css/app.css` (темы в отдельных файлах)
- Порядок функций — `functions.json`
- После правок достаточно обновить страницу (Ctrl+Shift+R для сброса кеша)

## Публикация на GitHub Pages

1. Убедиться, что в корне есть `.nojekyll` (уже есть)
2. В репозитории GitHub: Settings → Pages → Branch: main / root
